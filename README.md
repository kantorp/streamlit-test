# SHRN.TO — Market Press Intelligence

AI-powered tool for analyzing press releases and capital market reports.

## Features

- **Upload** — Upload PDF, DOCX, or TXT files with press releases and financial reports
- **Brief** — AI-generated structured brief with key information and identified trends
- **Export** — Export the generated brief to PDF, DOCX, or plain text

## Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```
